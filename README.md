# 📊 Finance Dashboard (Power BI)

## 🔍 Overview

This project is an interactive **Finance Dashboard** built using Power BI to analyze key business metrics such as **sales, profit, and discounts** across countries, products, and time.

The dashboard helps in identifying trends, comparing performance, and supporting data-driven financial decisions.

---

## 🎯 Objectives

* Analyze **sales performance over time**
* Track **profit generation**
* Evaluate **discount impact on revenue**
* Compare performance across **countries and products**

---

## 📌 Key Insights

* 💰 Total Profit: **3.53M**
* 📉 Discounts peaked significantly in **2014 compared to 2013**
* 📊 Sales show a **declining trend across months**
* 🌍 Certain countries contribute disproportionately to discounts
* 🛍️ Products like **Paseo** generate the highest profit share

---

## 📈 Dashboard Features

### 1. Country Filter (Slicer)

* Allows dynamic filtering by:

  * Canada
  * France
  * Germany
  * Mexico
  * USA

### 2. KPI Cards

* Total Profit
* Total Sales (monthly aggregated)

### 3. Visualizations

* 📊 **Bar Chart** → Sales by Month
* 📉 **Line Chart** → Sales Trend Over Time
* 🌍 **Bar Chart** → Discounts by Country
* 📅 **Bar Chart** → Discounts by Year
* 🥧 **Pie Chart** → Profit by Product
